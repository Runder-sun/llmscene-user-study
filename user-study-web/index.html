You will evaluate 5 scenes, each containing results generated by 5 different methods.

For each scene, you need to complete two ranking tasks:

  1. Physical Plausibility Ranking: Check for object overlap or out-of-bounds issues
  2. Visual Quality Ranking: Consider scene reasonableness, layout aesthetics, and accessibility

üìä Evaluation Criteria

Physical Plausibility

  • Object overlap / collision
  • Objects out of room bounds
  • Floating objects

Visual Quality

  • Rationality (Ra.): Appropriate room size, reasonable number of objects, good physical plausibility, and contains key objects for the room type
  • Spatial Layout (Spa.): Reasonable functional zoning and good spatial distribution balance. Rooms with objects overly clustered together or with large unused empty areas are considered poorly laid out
  • Accessibility (Ac.): All objects should be reachable and touchable, with sufficient walking space between objects for human movement. Overcrowding that makes objects inaccessible is unacceptable
