🏠 3D室内场景生成质量评估

研究说明

感谢您参与本次用户研究！本研究旨在评估不同AI方法生成的3D室内场景的质量。

⚠️ 重要提示

请在做排名决定之前仔细阅读提示文本！

提示描述了预期的房间设计，请考虑每种方法对提示要求的满足程度。

📋 任务概述

您将评估 5个场景，每个场景包含6种不同方法生成的结果。

对于每个场景，您需要完成两项排名任务：

  1. 物理合理性排名：检查物体重叠或越界问题
  2. 视觉质量排名：考虑场景合理性、布局美观性和可达性

📊 评估标准

物理合理性

  • 物体重叠 / 碰撞
  • 物体超出房间边界
  • 悬浮物体

视觉质量

  • 合理性 (Ra.)：适当的房间大小、合理的物体数量、良好的物理合理性，以及包含该房间类型的关键物体
  • 空间布局 (Spa.)：合理的功能分区和良好的空间分布平衡。物体过度聚集或存在大片空白区域的房间被认为布局较差
  • 可达性 (Ac.)：所有物体应可到达和触及，物体之间有足够的行走空间供人移动。过度拥挤导致物体无法到达是不可接受的

⏱️ 预计时间

整个评估过程大约需要 5-10分钟。
